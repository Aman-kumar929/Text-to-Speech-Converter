# AI Text-to-Speech Converter

A professional **Text-to-Speech (TTS) web application** built using **Flask** and **Microsoft Edge Neural Voices**.  
This app converts written text into **high-quality, natural-sounding speech** with support for
**multiple languages, voice gender selection, pitch, and speed control**.

---

## Features

-  Multiple language support (English, Hindi, French, German)
-  Male & Female AI voices
-  Control speech pitch and speaking rate
-  High-quality Edge Neural voices
-  Download generated audio (MP3)
-  Modern UI with video background
-  Fully responsive professional design

---

## Tech Stack

- **Backend:** Python, Flask  
- **Text-to-Speech Engine:** Microsoft Edge TTS  
- **Translation:** Google Translator  
- **Frontend:** HTML5, CSS3
